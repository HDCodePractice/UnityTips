# 参与提交说明

* 请将你的项目放在对应的目录中，如果有多个项目，请通过项目名称来区别，项目名称请加上 - username
* 提交项目时也请同时更新所在目录中的README.md文件
* 在发布WebGL版本时，请在 ***Player Settings*** 中的 ***Publishing Settings*** 里将 ***Compression Format*** 设置为 ***Disabled***
